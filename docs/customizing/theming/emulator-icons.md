# Adding Emulator Icons

Create a `.media` directory under the corresponding folder to create icons for Emulators, Collections and Tools.

For example:

```
SD_CARD
├─ .media/
│  ├─ Collections.png
│  ├─ Recently Played.png
├─ Collections/
│  ├─ .media/
│  │  ├─ Collection 1.png
│  │  ├─ Collection 2.png
├─ Roms/
│  ├─ .media/
│  │  ├─ GBA.png
│  │  ├─ SFC.png
│  │  ├─ Custom Emulator Name (GBC).png
│  │  ├─ etc...
├─ Tools/
│  ├─ .media/
│  │  ├─ tg5040.png
```
